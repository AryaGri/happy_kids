Для корректного отображения кириллицы в PDF-файлах назначений
скачайте шрифт DejaVuSans.ttf и поместите его в эту папку.

Скачать: https://github.com/dejavu-fonts/dejavu-fonts/releases
(файл DejaVuSans.ttf из архива ttf)

На Windows также используется системный шрифт Arial (C:\Windows\Fonts\arial.ttf).
