# Установка приложения «Платформа»

Инструкция для пользователей.

## Требования

- **Windows 10 или 11**
- Подключение к интернету

---

Если у вас папка проекта с bat-файлами:

1. **Установите Python** с [python.org](https://www.python.org/downloads/) (поставьте галочку «Add Python to PATH»)
2. Откройте **PowerShell** или **Командную строку** в папке проекта
3. Выполните:
   ```
   pip install -r desktop_client/requirements.txt
   .\run_desktop.bat
   ```
4. Или дважды щёлкните **`run_desktop.bat`** (если Python в PATH)
