# Десктопное приложение

Одно приложение для всех ролей. Веб-интерфейс 1:1 — тот же вид, что в браузере.

## Установка

```bash
cd desktop_client
pip install -r requirements.txt
```

## Запуск

```bash
.\run_desktop.bat
```
или
```bash
python desktop_client/main.py
```

## Как пользоваться
1. Экран настроек: укажите адрес сервера (по умолчанию https://happy-kids-iu6.ru), нажмите «Проверить подключение», затем «Войти».
2. Откроется форма входа. Войдите под своей учётной записью.
3. После входа откроется панель в зависимости от роли: администратор, врач, родитель или ребёнок.
4. Адрес сервера сохраняется в `desktop_client/app_config.json`.

## Последовательность запуска
1. Запустите Django-сервер: `python manage.py runserver`
2. Запустите приложение: `.\run_desktop.bat` или `python desktop_client/main.py`

## Сборка для раздачи (скачать .exe)
```batch
.\build_desktop.bat
```
Результат: папка `dist/Платформа/`. Заархивируйте её в ZIP — пользователи распаковывают и запускают `Платформа.exe`. Подробнее: `desktop_client/BUILD.md`
