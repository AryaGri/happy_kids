# Сборка приложения для раздачи

## Требования
- Python 3.10+
- Windows 10/11 (pywebview использует Edge WebView2, встроенный в систему)

## Сборка

Из корня проекта:
```batch
.\build_desktop.bat
```

Или вручную:
```batch
pip install pyinstaller
cd c:\Users\ayagr\my_project
pyinstaller --noconfirm desktop_client\platform.spec
```

## Результат
Папка `dist/Платформа/`:
- `Платформа.exe` — запуск приложения
- DLL и библиотеки рядом

## Раздача пользователям

1. **ZIP-архив** — заархивируйте папку `dist/Платформа`:
   ```
   dist/Платформа/
     Платформа.exe
     _internal/
     ... (остальные файлы)
   ```
   Пользователь распаковывает и запускает `Платформа.exe`.

2. **GitHub Releases** — загрузите ZIP как релиз (например, `Платформа-Windows.zip`).

3. **Свой сервер/сайт** — положите ZIP в раздел загрузок.

## Важно для пользователей
- Windows 10/11 (WebView2 обычно уже есть)
- Приложение подключается к серверу — укажите адрес вашего сервера при первом запуске
- Не требует установки Python
